# Chinook Digital Music Store Analysis

An end-to-end data analysis project leveraging Python, SQL, and Pandas to query business insights from the sample Chinook SQLite database. This project explores track sales, regional revenues, and staff performance metrics.

## 📊 Business Questions Addressed
1. What are the top 10 best-selling tracks globally?
2. Which country generates the highest total revenue?
3. Who is the top-performing sales support employee?

## 🛠️ Tech Stack & Architecture
* **Language:** Python
* **Database Engine:** SQLite (`sqlite3`)
* **Data Manipulation:** Pandas
* **Data Visualization:** Matplotlib
* **Environment:** Jupyter Notebook / VS Code
